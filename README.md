## Dependencies

poetry >= 1.8.3

```bash
curl -sSL https://install.python-poetry.org | python3
```

## How it works

Clone repository: </br>
```bash
https://github.com/audreyteles/who-face.git
```

Activate environment: <br>

```bash
poetry shell
 ```

Install libs: <br>

```bash
 poetry install
 ```

Start application: <br>

```bash
python main.py
```

### Commands

q - quit

p - take a picture

r - recognize face
